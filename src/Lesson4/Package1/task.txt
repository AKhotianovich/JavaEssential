Используя IDEA, создайте проект. Требуется:
Создайте абстрактный класс AbstractHandle.
В теле класса создать абстрактные методы void open(), void create(), void change(), void save().
Создать производные классы XMLHandle, TXTHandle, DOCHandle от базового класса AbstractHandle.
Написать программу, которая будет выполнять определение документа и для каждого формата должны
быть методы открытия, создания, редактирования, сохранения определенного формата документа.
